# Memory Game


## Description:
Memory Game is an interactive web-based game built with HTML, CSS, and JavaScript, where players flip cards to find matching pairs.
This project demonstrates DOM manipulation, event handling, state tracking, and basic game logic, while creating an engaging user experience.


## Tech Stack:
- HTML5 
- CSS3 
- JavaScript 
- Git & GitHub 
- GitHub Pages 


## How to Play:
1. Click on a card to reveal its symbol.
2. Click a second card to find a matching pair.
3. If the two cards match, they remain revealed. Otherwise, they flip back.
4. Continue until all pairs are matched.
5. Click Reset to start a new game.


## Features:
- Interactive card-flipping gameplay
- Automatic matching detection
- Reset functionality to restart the game
- Visual feedback on matched pairs
- Clean, minimalistic UI for focus and engagement
